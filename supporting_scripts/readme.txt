# Key scripts
- step1_run_models.sh --> produce REGENIE Step1 models
- step2_run_gene_tests_pgen.sh --> produce REGENIE Step2 SKAT-O results
- step2_run_single_tests_pgen.sh --> produce REGENIE Step2 single variant results
- step2_run_gene_tests_conditional.sh --> produce REGENIE Step2 SKAT-O results with conditional
- process_regenie_results.R --> parse/filter REGENIE Step2 results
- gene_trait_known_replication.R --> script to measure overlap with known gene-trait assocaitions
- single_vs_burden_analysis.R --> script to classify associations as single, single-group or group
